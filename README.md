# Primal-Dual Multi-Agent Off-Policy Actor-Critic for Distributed Reinforcement Learning

This repository contains the implementation of an off-policy multi-agent reinforcement learning (MARL) algorithm using Primal-Dual methods for Consensus.

Authors: Shrinidi Kuppurajan and Om Mahajan

## Environments supported:

- [StarCraftII (SMAC)](https://github.com/oxwhirl/smac)
- [Multiagent Particle-World Environments (MPEs)](https://github.com/openai/multiagent-particle-envs)
